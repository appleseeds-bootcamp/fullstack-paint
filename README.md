A simple paint app

## Tech Stack

JS - React<br>
Python - Bottle<br>
DB - pymysql (MySQL)<br>

### Build App and Install Depedencies
Back End:
```
cd paint-be
python -m venv venv
pip install -r requirments.txt
```
FE:
```
cd paint-fe
npm install
npm run build
```

### Launch app
```
cd paint-be
python main.py

visit http://localhost:8080 on your nearest browser
```
